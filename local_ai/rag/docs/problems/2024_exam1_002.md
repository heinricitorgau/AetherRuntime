Write a program that reads n and generates an arrow pattern. For n=3, the displayed rows are 3, then 2 2, then 12321, then 2, then 1.

Required features:
  - Read integer input n
  - Generate the arrow pattern using loops
  - Print the center row with ascending then descending digits
  - Print upper and lower rows with correct spacing

Sample input:
3

Expected output contains: 3, 2, 12321, 1
