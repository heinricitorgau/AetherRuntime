(a) [3 pts] Read and display 4 points

    Area formula: √(s(s-a)(s-b)(s-c)) where s=(a+b+c)/2
    Show message if triangle cannot be formed

(d) [9 pts] Check if 4th point is inside or outside the triangle
    Hint: Point is inside if sum of areas of 3 sub-triangles equals original triangle area

Required features:
  - Read 4 coordinate pairs
  - Calculate distances
  - Compute triangle area using Heron's formula
  - Check point-in-triangle
  - Handle edge cases

Sample input:
0 0 0 3 4 0 1 1

Expected output contains: points, distance, area, perimeter
