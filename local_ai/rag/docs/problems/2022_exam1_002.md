Write a program that reads n and generates a diamond pattern.

Example with n=3:
1
2 2
32123
2 2
1

The pattern forms a diamond shape with numbers increasing towards middle.

Required features:
  - Read integer n
  - Generate top half (expanding)
  - Generate middle row
  - Generate bottom half (contracting)
  - Correct number placement and spacing

Sample input:
3

Expected output contains: 1, 2 2, 32123, 2 2, 1
