Write a program to simulate a tug war game using stone, scissors, and paper. Print a welcome message, read the winning distance n, display positions from -n to n with a flag at the current distance, read the player's play, generate the computer's play, decide the round winner, update the tug status, and continue until either side reaches the winning distance.

Required features:
  - Read winning distance n
  - Display the tug-war track from -n to n
  - Read player choice for stone, scissors, or paper
  - Generate or otherwise select a computer choice
  - Decide draw/player win/computer win
  - Update the current distance
  - Loop until there is a winner
  - Print a final win or lose message

Sample input:
1
1

Expected output contains: Tug, War, choice
