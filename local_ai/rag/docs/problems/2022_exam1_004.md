Simulate a tug war game with rock-paper-scissors.

(a) [4 pts] Print welcome and read winning distance n

(b) [8 pts] Display distance scale from -n to n with flag at current position

(c) [6 pts] Read player choice, generate random computer choice (1=rock, 2=scissors, 3=paper)

(d) [5 pts] Determine winner and update tug position

(e) [5 pts] Continue until someone reaches ±n and announce winner

Winning rules: rock beats scissors, scissors beats paper, paper beats rock

Required features:
  - Display tug war board
  - Read player input (1,2,3)
  - Generate random computer choice
  - Implement win logic
  - Move flag based on outcome
  - Loop until winner

Sample input:
3\n1\n1\n1

Expected output contains: Welcome, Tug War, Enter, winner
