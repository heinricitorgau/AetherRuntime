Write a function that receives n and returns the sum of the series 1 - 2/3 + 6/7 - ... + n!/(n^2 - n + 1). Write main to read n, call the function, and print f(n). Example: n=11 gives f(11)=324096.52032087732000000000.

Required features:
  - Read integer input n
  - Use a function to compute the series
  - Compute factorial terms
  - Alternate signs correctly
  - Print the value of f(n) with decimal precision

Sample input:
11

Expected output contains: 324096
