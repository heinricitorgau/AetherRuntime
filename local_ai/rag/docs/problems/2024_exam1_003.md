Write a program that simulates a roulette-style guessing game. The player's initial point is 0. Show choices O/E/L/C/R/N for odd, even, left, center, right, or exact number. The program generates a random winning number from 1 to 36, asks for the points at risk, applies the winning rates, asks whether to continue, and finally prints total guesses and the player's point total.

Required features:
  - Print the guessing choices table
  - Read the player's choice
  - Read the points at risk
  - Generate or select a winning number from 1 to 36
  - Handle odd/even, left/center/right, and exact-number guesses
  - Update the player's point total using the risk/winning rate
  - Ask whether to continue
  - Print total number of guesses and final points

Sample input:
E
20
N

Expected output contains: choice, point, winning
