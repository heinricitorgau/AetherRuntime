Write a program that reads n and generates a triangle pattern.

Example with n=3:
3
2 2
11111
222
3

Pattern expands with each row and displays repeated digits.

Required features:
  - Read integer n
  - Generate expanding triangle
  - Each row i has i digits (possibly repeated)
  - Correct spacing
  - Handle n from 1 to 10

Sample input:
3

Expected output contains: 3, 2 2, 11111, 222, 3
