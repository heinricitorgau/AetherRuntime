Write a program that reads n and calculates the sum of the series:
1 - 2/3 + 6/7 - ... + n!/(n²-n+1)

Formula: Sum from i=1 to n of: (-1)^(i+1) * i! / (i²-i+1)

Example: When n=11, f(11) = 324096.52032087732

Required features:
  - Implement factorial calculation
  - Calculate series with alternating signs
  - Compute each term's denominator: i²-i+1
  - Sum all terms
  - Output with decimal precision

Sample input:
11

Expected output contains: 324096
