Write a program with the following geometry functions:

(a) [4 pts] Function: area of triangle formed by line ax+by=c with x-axis and y-axis
    Hint: When y=0, x=c/a. When x=0, y=c/b

(b) [5 pts] Function: distance from point (x,y) to line ax+by=c
    Formula: |ax₀+by₀-c|/√(a²+b²)

(c) [8 pts] Function: area of triangle formed by three points
    Use Heron's formula: Area = √(s(s-a)(s-b)(s-c)) where s=(a+b+c)/2
    Return 0 if triangle cannot be formed

(d) [5 pts] Function: show line equation passing through two points

Required features:
  - Implement 4 geometry functions
  - Handle floating point calculations
  - Calculate distances between points
  - Apply Heron's formula
  - Validate triangle formation

Sample input:
0 0\n0 3\n4 0

Expected output contains: area, distance, triangle
