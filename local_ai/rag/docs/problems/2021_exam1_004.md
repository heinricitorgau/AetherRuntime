Write a program that simulates a betting/guessing game.

Game board (1-12):
- Low: 1-4
- Mid: 5-8
- High: 9-12

Betting options with risk:win ratios:
- Odd|Even: 1:2
- Low|Mid|High: 1:3
- Single Number: 1:12

Game flow:
(a) [5 pts] Choose among 6 possible guesses
(b) [5 pts] Generate random number 1-12
(c) [10 pts] Award points based on risk:win ratio if correct
(d) [2 pts] Allow continue playing
(e) [4 pts] Display final points when quit

Required features:
  - Read betting choice
  - Generate random number 1-12
  - Calculate points based on risk:win ratio
  - Maintain game state and points
  - Loop until quit
  - Display results

Sample input:
middle\n20\ny\nno

Expected output contains: Enter, points, choice
