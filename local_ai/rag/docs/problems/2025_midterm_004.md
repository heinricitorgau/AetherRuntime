Simulate an even/odd guessing game:

(a) [3 pts] Generate 5 different random numbers (1-10), hide them

(b) [10 pts] Game loop:
    - Display hidden numbers as asterisks
    - Player picks position
    - Player guesses even or odd
    - If correct, player wins 5 points
    - Reveal the number

(c) [4 pts] Continue until all numbers revealed
    Display final score

Game ends when all 5 positions are revealed.

Required features:
  - Generate 5 unique random numbers (1-10)
  - Hide numbers initially
  - Track guessed positions
  - Calculate correctness
  - Maintain score
  - Display game state

Sample input:
2\nO\n5\nE

Expected output contains: Numbers, win, points, Pick
