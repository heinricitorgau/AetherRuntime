Write a program that reads n and generates an arrow pattern.

Example with n=3:
3
2 2
12321
2
1

The pattern forms an arrow pointing upward and then downward.

Required features:
  - Read integer n
  - Generate top half of arrow (expanding)
  - Generate bottom half of arrow (contracting)
  - Correct spacing and number patterns
  - Handle n from 1 to 10

Sample input:
3

Expected output contains: 3, 2 2, 12321, 2, 1
