Write a program that reads n and calculates the sum of the series:
1 - 5/3 + 7/8 - ... + (2n+1)/(n²-1)

Formula: Sum from i=1 to n of: (-1)^(i+1) * (2i+1) / (i²-1)
(The i=1 term is defined as 1; start accumulating from i=2 to avoid dividing by zero.)

Example: When n=10, sum = -0.236724

Required features:
  - Read integer n
  - Calculate numerator: 2i+1
  - Calculate denominator: i²-1
  - Apply alternating signs
  - Sum all terms
  - Output result with precision

Sample input:
10

Expected output contains: -0.237
